# わかったこと
```
Schemaの体裁を整える
npx prisma format
```
```
Prismaクライアントの生成、TypeScriptの型情報の生成（型などを修正したらまたコマンドを入力）
npx prisma generate
```
```
schema.prisma変更した際に新しいマイグレーションファイルを作成（その前にdockerなどでDBを起動する）
npx prisma migrate dev（開発用）
npx prisma migrate deploy（本番用）
```
```
テーブルを一度削除し再生成(開発時の方がいい)
npx prisma migrate reset
```
