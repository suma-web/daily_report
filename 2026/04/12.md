- prismaの組み込み流れ
```
npm install prisma --save-dev    // prismaインストール
npm install @prisma/client
↓
npx prisma init     // 初期化
↓
DBの設定（.env）
↓
schemaを書く（テーブルの作成）
↓
npx prisma migrate dev --name init
npx prisma migrate dev --name 〇〇（特定のカラムに変更があれば〇〇にカラム名を挿入するのが理想）

--name init              // 初期作成
--name add_user    // user追加
--name add_todo.    // todo追加

↓
npx prisma generate（特定のカラムに変更があれば）
↓
npx prisma studio     // prisma studioで反映確認
```
