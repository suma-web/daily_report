- JavaScriptモジュールはHTMLから`<script type="module">`で読み込むことができますが、script要素ごとに別々のモジュールスコープを持ちます。 
```
<script type="module">
    export const scopeA = "A";
  </script>
  <script type="module">
    // 異なるmoduleスコープの変数には直接アクセスできない
    console.log(scopeA); // => ReferenceError: scopeA is not defined
  </script>
```
