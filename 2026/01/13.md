# わかったこと
```
const fruits = ["りんご", "みかん"];
console.log(fruits.join(“,”));  //りんご, みかん（一文のように繋げることができる)
```
```
const numbers = [1, 2, 3, 4];
numbers.every((num) => num >0);     //true     全ての配列を関数と比較し、全てが当てはまればtrue
numbers.every((num) => num >1);     //false

numbers.sum((num) => num >3);     //true     全ての配列を関数と比較し、ある程度当てはまればtrue
numbers.sum((num) => num >5);     //false    

numbers.filter((num) => num >=3);    //[3, 4]

number.map((num) => num*3);     //[3, 6, 9, 12]  各配列ごとに処理し、その結果を格納する
```
エラー
```
try{
   …;
} catch (e) {
    alert(e);
}
```

- eはエラーオブジェクトで、ポップアップがブラウザに表示される
- 変数と定数の使い分けは、値の書き換えを行わない場合はconstを使う。
- verは一つ前のバージョンの変数・定数

特殊なオブジェクト（JavaScriptであらかじめ定義されている機能）
```
window　//webブラウザ全体のオブジェクト

document  //現時点で表示されいるページ全体のオブジェクト
```

- 何度も使用するものは変数・定数に設定する

- HTMLのオブジェクトを持ってくる場合、「$変数名」と記入するのが暗黙のルール
