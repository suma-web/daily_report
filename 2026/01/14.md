# 感じたこと
- 読書感想
    - 本書籍は、生成AI（Chat GPT）とコーディングとの向き合い方を叙事的に描いたもの。一番心に響いたのは「Chat GPTは使用者以上の力を発揮することはない」。つまりは、使用者のステータスによってプロンプト文の解像度も変わり、回答もその基準でかえってくること。
    - そして、出力される結果が多様なことは悪いことではなく、視点を広げるきっかけになること。
- Django
    - メールを送信するモジュールがうまくいかない。まだ言語化が難しいが、送信にあたって初期設定に障壁を感じる。
- JavaScript
