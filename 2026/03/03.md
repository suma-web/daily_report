```
型の確認
s := 100
fmt.Printf(“%T\n”, s)  // int

Var f string = "Hello”

```

TypeScript
- TypeScriptは静的型付け言語。JavaScriptは動的型付け言語。
- 型が予想できる場合は明示的に記載しなくてもいい。`any`は使わない。
```
プロジェクト初期化（package.jsonをインストール）
npm init -y

TypeScriptインストール
npm install --save-dev typescript

tsconfig.jsonの生成
npx tsc --init

コンパイル（JavaScriptに変更）
npx tsc
npx tsc --watch      // 逐次コンパイルするのは大変なので代案

※ package.json に追加：
"scripts": {
"dev": "tsc --watch",
"build": "tsc"
}

ターミナルで起動
npm run dev
```
もしくは
```
distファイル不要でコンパルできる
npm install --save-dev ts-node

実行
node dist/index.js
```
```
階層イメージ
my-project/
├─ node_modules/     // TypeScriptインストール時に生成
├─ package.json
├─ tsconfig.json
├─ src/                   // mkdirで作る。開発時の編集。
│ └─ index.ts        
├─ dist/           // mkdirで作る。出力ファイル。（主にnpx tscを使用する場合）
```
