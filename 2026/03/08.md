# わかったこと
- スライス型
サイズ未指定、またはmake()を使用するとスライスが生成される。
```
slice0 := []int {1, 2, 3}
slice1 := make([]int, 3, 5)

配列型：
array0 := [3]int {1, 2, 3}           // []に3があり要素数が記載
array1 := [...]int {1, 2, 3, 4}
```
```
sl := []int{100, 200}
sl = append(sl, 300)    // [100, 200, 300]
len(sl)      // slの要素数
cap(sl)    // slの占有メモリ数。メモリーを気にするような開発の場合は、容量にも気をつけます。
copy(sl)  // 
```
