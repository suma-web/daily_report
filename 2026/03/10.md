チャネル（データの送受信）
```
	var ch1 chan int
	ch1 = make(chan int, 1) 
	fmt.Println(ch1)
	fmt.Println(cap(ch1))
	fmt.Println(len(ch1))

	ch1 <- 1
	ch1 <- 2        // ch1は要素数1までなのでデッドロックになる
```
- `chan 型（任意の型）`
```
	ch1 := make(chan int, 3)
	ch1 <- 1
	ch1 <- 2
	ch1 <- 3
	close(ch1)                  // ないとデッドロックになる
	for i := range ch1 {
		fmt.Println(i)
	}
```
- 複数のチャネルの送受信に、ゴルーチンを停止させることなく制御できる

スコープ（パッケージをインポートする）
main.go
```
package main

import (
	"fmt"
	"golang/foo"     // main.goと同じ階層にgo.mod 作成（コマンド：go mod init golang）
)

func main() {
	fmt.Println(foo.Max)    // 100
	fmt.Println(foo.ReturnMax())   // 100 1 
}
```
foo/foo.go
```
package foo      // package フォルダ名（パッケージ名）

const (
	Max = 100    // 大文字であればインポートできる
	min = 1
)

func ReturnMax() int {   // 大文字であればインポートできる
	return Max
}
```
- インポートされたパッケージはアツファベット順
- パッケージのテストができる（パッケージ名は`xxx_test.go`とする）

Generics（1.18 ver から）
構造が同じ関数に使える
```
Func PrintSlice[T any](s []T) {
	for _, v := range s {
		fmt.Println(v)
	}
}

Func main() {
	PrintSlice[int]([]int{1, 2, 3, 4, 5, 6)              
	PrintSlice[string]([]string(“a”, ”b”, ”c”))
}   //  [int]や[string]は型推論で波線が表示される時があるので、その時は省略可能
```
