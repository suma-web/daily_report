# わかったこと
- `const`と`let`を使う。

|タイプ|再宣言|再代入|スコープ|初期化|
|---|---|---|---|---|
|let|×|⚪︎|ブロック|×|
|const|×|×|ブロック|×|
|var(非推奨)|⚪︎|⚪︎|関数|undefined|

- JavaScriptは動的型付け言語なので、暗黙的に型が変換する。
- プリミティブ型の値を渡す場合には値がコピーされる。オブジェクトの場合は参照が渡されるので、その渡した先の関数内でオブジェクトのプロパティを変更した場合は、元のオブジェクトにも影響が出る。
- 分割代入
- オブジェクト同士の比較は特定のプロパティを用いる必要がある。プリミティブ型は値の比較、オブジェクト型は参照の比較。
```
function greeting(name, say){
    say = say || 'hello';
    console.log(`${say},${name}`);
}

greeting("Bob", "hi");
greeting("Bob");
```
・初期化の状態で「undefined」を設定することはない。
・関数は実行可能なオブジェクト
・コールバック。関数を引数にし関数を呼び出すこともできる。無名関数を呼び出すこともできる。
