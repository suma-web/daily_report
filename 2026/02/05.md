# わかったこと
- `console.log`のみだと`setTimeout`を通さずすぐにコンソールログに表示されてしまうため、1秒後に表示できない。
```
＜目的が果たせない＞
function hello(name) {
    console.log(`hello ${name}`)
}

setTimeout(hello('Tom'), 1000);
```

```
＜関数をそのまま渡すのが正解＞
function hello(name) {
    return () => {
         console.log(`hello ${name}`)
    }
}

setTimeout(hello('Tom'), 1000);
```
- コンストラクター関数において `.prototype`をつけ、メソッドを追加させる。
- 演算子の優先順位（グループ化が高い）

- `for`文を`while`文で記述する方法
```
const array = [1, 2, 3, 4, 5];

let v, i = 0;
while( v = array [i++] ){        //for文のi++を配列の[ ]内に記載
   console.log(v);
}
```
- 以下の2点を満たすオブジェクトを、イテレータ と言う。
    - `.next()` メソッドを持つ こと
    - `.next()` を実行するとイテレータリザルトを返す こと
- イテレータを簡略化したのがジェネレータ
