# わかったこと
- 個別でCSSファイルを読み込みたい時、<head></head>の中に{% block customcss %}を追加する
- フォームで情報を送る場合は『post」、それ以外は「get」と覚える
- redirectとrenderの違い
    - Renderはリクエストを受けたら、表示だけリクエストされたURLに飛ぶ
    - Redirectはリクエストを受けたら、表示だけではなくURLもそのリクエストされたURLに飛び
- {% load static %} 静的ファイルを読む込むためのおまじない（CSSなど）
